2.3 Events

NEEDS FIXING!

HTML events can be defined as something that the browser or user does which happen to HTML elements. When an event happens, you usually want to do something. Here is a browser event example: a HTML page which has finished loading, here is a user event example: a button was clicked. Both of these are events, and most of the time things will happen when these events are triggered, such as rendering the page or executing some code.

So where does JavaScript fit in? JavaScript lets you execute code when events are detected. and HTML allows event handler attributes, with JavaScript code, to be added to HTML elements.

Event handlers can be used to handle, and verify, user input, user actions, and browser actions:

- Things that should be done every time a page loads
- Things that should be done when the page is closed
- Action that should be performed when a user clicks a button
- Content that should be verified when a user inputs data and much more...

Different methods can be used to let JavaScript work with events:

- HTML event attributes can execute JavaScript code directly
- HTML event attributes can call JavaScript functions
- You can assign your own event handler functions to HTML elements and much more...

List of Events

Here is a trimmed down list of some events:


onchange: An HTML element has been changed
onclick: The user clicks an HTML element
onmouseover: The user moves the mouse over an HTML element
onmouseout: The user moves the mouse away from an HTML element
onkeydown: The user pushes a keyboard key
onload: The browser has finished loading the page
dblclick: The event occurs when the user double-clicks on an element
drag: The event occurs when an element is being dragged
drop: The event occurs when the dragged element is dropped on the drop target
load: The event occurs when an object has loaded
mousedown: The event occurs when the user presses a mouse button over an element
mouseover: The event occurs when the pointer is moved onto an element, or onto one of its children
mouseup: The event occurs when a user releases a mouse button over an element
resize: The event occurs when the document view is resized
scroll: The event occurs when an element's scrollbar is being scrolled
show: The event occurs when a <menu> element is shown as a context menu

General Syntax

Dot notation (.) is used to access the properties of the DOM, preceded by document and followed by the property or method name.


    // Add text to paragraph with id "last_paragraph"
    // Here we used the getElementById() method 
    // followed by the changing the innerHTML property value.
    document.getElementById("last_paragraph").innerHTML = "Hello, I was just added!";

Examples

Change this text's color, size and font!


    document.getElementById("change_css").style.color = "blue";
    document.getElementById("change_css").style.fontFamily = "Arial";
    document.getElementById("change_css").style.fontSize = "64";

Write new text to document!


    // Use createElement method
    var el = document.createElement("H3");
    // Append to the document's body
    document.body.appendChild(el);
    // Create a text node
    var text = document.createTextNode("Hello there, I wasn't here before!");
    // Append node to el
    el.appendChild(text);
