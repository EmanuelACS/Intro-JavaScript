1.2 Operators

Math Operators

Math operators perform mathematical calculations.

+ addition of two numbers and concatenates strings
- subtraction of two numbers and unary operator to negate operand
/ division of two numbers
* multiplication of two numbers
% modulus (the number left over after division of 2 numbers)
++ increment (+1)
-- decrement (-1)

Comparison Operators

Comparison operators return a value of "true" or "false"

== Returns true if the operands are equal.
=== Returns true if the operands are equal and of the same type.
!= Returns true if the operands are not equal.
> Returns true if the left operand is greater than the right operand.
< Returns true if the left operand is less than the right operand.
>= Returns true if the left operand is greater than or equal to the right operand.
<= Returns true if the left operand is less than or equal to the right operand.

Logical Operators

Logical operators that compare the right and left side values then return true or false.

&& (and) - Returns true if both sides of the argument are true
|| (or) - Returns true if either side of the argument is true
! (logical negative) - Returns the opposite of the current boolean value. Turns true to false, and false to true.

Assignment Operators

Assignment operators apply the right side value to the left side variable. In combination with some math operators, a calculation can be done first, with the result assigned to the left side value.

= assigns the value of the right side to the left side of the operator
+= addition assignment, adds both arguments and returns the value to the left side
-= substraction assignment, substracts both numbers and assigns the result to the left side
*= multiplies and assigns to the left side argument
/= divides and assigns to the left side argument
%= modulus result and assigns to the left side argument

String Operators

String operators work on string values similarily to how math operators work on numbers.

+ concatenates two strings together
+=concatenates two strings and assigns result to left side argument

Specials ternary

Short forms of if, then, else expression. It is normally used for simple things as it is hard to read. It requires three arguments to work.

? : - a simple if, then, else expression. The test is on the left side of the question mark, with the true (then) action placed between the" ?" and the ":" and the false (or else) action on the right of the ":"

General Syntax

The operator is placed between the two arguments, (variables, expressions, numbers, strings, etc.), that it is going to act upon. In general, the value on the right is assigned or compared to the value on the left of the operator.
