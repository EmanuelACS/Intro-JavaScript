1.3 Conditional Operators

A "conditional statement" is a set of rules performed if certain conditions are met. It is used when wanting to execute certain code only when the correct conditions are met.

Operators


Use if to specify a block of code to be executed, if a specified condition is true.
Use else to specify a block of code to be executed, if the same condition is false.
Use else if to specify a new condition to test, if the first condition is false.
Use switch to specify many alternative blocks of code to be executed.

"If" Statement

The if statement comes at the start and has the following syntax: 


    if (condition) {
        // block of code to be executed when the condition is true
    }

"Else" Statement

The else statement comes after all if or if else statements have been executed and has the following syntax: 


    if (condition) {
        // block of code to be executed when the condition is true
    } else {
        // block of code to be executed when the condition is false
    }

"Else If" Statement

Any else if statements come right after the previous if or else if statements, but before the else statement and has the following syntax: 


    if (condition1) {
        // block of code to be executed when condition1 is true
    } else if (condition2) {
        // block of code to be executed when condition2 is true and condition1 is false
    } else {
        // block of code to be executed when condition 1 & 2 are false
    }

Example

By clicking on the "Try me!" button below, you will test the example below.


    // ask user to choose a number
    var question = prompt("Choose either number 1, 2 or 3");
    if (question == 1) {
        // alert user of his choice when user enters 1
        alert("You have chosen the number 1!");
    } else if (question == 2) {
        // alert user of his choice when user enters 2
        alert("You have chosen the number 2!");
    } else if (question == 3) {
        // alert user of his choice when user enters 3
        alert("You have chosen the number 3!");
    } else {
        // block of code to be executed when user doesn't enter 1, 2, or 3
        alert("You have not chosen 1, 2 or 3!");
    }

"Switch" statement

The switch statement is used to perform different actions based on different conditions, and uses the keyword break to break out of the loop. It has the following syntax: 


    switch(expression) {
        case a:
            // block of code to be executed if expression == a
            break;
        case b:
            // block of code to be executed if expression == b
            break;
        default:
            // default block of code to be executed if not case a or b
            // this block statement doesn't need a break;
    }

Example

By clicking the "Try me!" button, you will test the switch statement.


    // ask user to choose a number
    var question = prompt("Choose a number between 1 and 3")
    switch(question) {
        case 1:
            // block of code to be executed if user chooses 1
            alert("You have chosen the number 1!");
            break;
        case 2:
            // block of code to be executed if user chooses 2
            alert("You have chosen the number 2!");
            break;
        case 3:
            // block of code to be executed if user chooses 3
            alert("You have chosen the number 3!");
            break;
        default:
            // default code to be executed if user doesn't choose 1, 2 or 3
            alert("You have not chosen 1, 2 or 3!");
    }

Example using "if" and "switch"

Here is another example that uses both if and switch, click the button to try it!


    // ask user to choose a number between 1 and 10
    var question = prompt("Choose a number between 1 and 10")
    // if number is between 1 and 3
    if (question >= 1 && question <= 3) {
        // check which of them it is and tell the user
        switch (question) {
            case "1":
                alert("You have chosen the number 1!");
                break;
            case "2":
                alert("You have chosen the number 2!");
                break;
            case "3":
                alert("You have chosen the number 3!");
                break;
            default:
                alert("This should never run");
        }
    }
    // otherwise, if it is between 4 and 10 
    else if (question >= 4 && question <= 10) {
        // tell the user
        alert("You have chosen a number between 4 and 10!");
    }
    // if it isn't between 1 and 10 
    else {
        // tell the user his guess was invalid
        alert("Invalid guess!");
    }
