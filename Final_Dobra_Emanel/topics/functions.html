1.5 Functions

Functions in JavaScript are a block of code designed to perform specific tasks. A function is executed when something calls it.

Syntax

A function is defined with the function keyword, followed by their name, followed by parentheses (), followed by curly brackets { }.

Function names can contain letters, digits, underscores and dollar signs. Their parantheses contain the parameters (if any) separated by commas. The block of code to be executed is contained with the curly brackets.
Functions are used mainly for the ability to reuse code without having to redefine it again. 


    function_name(parameter1, parameter2..., parameterN) {
        // block of code to be executed
    }

Executing a function

Functions are executed when:
1. They are called from the code i.e myFunction();
2. An event occurs (such as user clicking a button)
3. They are self-invoked, occurring automatically 


    // called from the code
    function sayHi(userName) {
        alert("Hi " + userName + "!");
    }
    sayHi("Mike");

    // called when an event occurs
    onclick="sayHi("Mike");

    // self invoked function
    (function(){ 
        var nickName = "Mikey"; 
        alert(nickName); 
    })();

Function Return

When a return statement is reached, the function will stop executing. You can return a value, which will then be returned to its caller. 


    // function to return the name user inputs
    function getName() {
        return prompt("What is your name?");
    }

    // call the function and return its value to the caller
    var userName = getName();
