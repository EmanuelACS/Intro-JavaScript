1.9 String Object

A string built-in object is used to store and manipualte text. A JavaScript string is zero or more characters long written within " " quotes. The main string property is length which can be checked at any moment using string_name.length which returns the length of your string.

Syntax

Both single quotes and double quotes can be used to declare a string. Strings can also be declared as objects using the new keyword, however, it slows down the execution speed, so it is rarely used.


    // Single quotes
    var name = new String('my_name'); // typeof(name) is object
    var _name = 'my_name'; // typeof(_name) is string
    // Double quotes
    var name2 = new String("my_name2"); // typeof(name2) is object
    var _name2 = 'my_name2'; // typeof(_name2) is string
    // To have quotes within the string you cannot use the same quotes used to declare
    var x = ""Hello" is what he said" // will give an error
    var y = "'Hello' is what he said" // printing x will give 'Hello' is what he said
    // To check the length of a string
    var short = "19 characters long";
    // (short.length == 19) would return true

Escape Character

Previously we saw that we cannot have multiple identical quotes within a string declaration. However, a backslash character will enable us to do so. The backslash(\) character turns special characters such as " or ' into string characters. Try it!


    // Using backslashes in order to use single and double quotes 
    var nums = "The number \"3\" is written as \'three\'."
    alert(nums);

You can also break long lines of code using the backslash(\).


    var looong = "This is a very ver very very ... very very long string";
    // you can write the same this way
    var loong = "This is a very ver very \ 
    very ... very very long string";

String Methods

There are plenty of methods useful when manipulating strings. Here are a few:
charAt(): Returns the character at the specified index.
concat(): Joins two or more strings, and returns a new string.
fromCharCode(): Converts Unicode values to characters.
includes(): Checks whether a string contains the specified substring.
localeCompare(): Compares two strings in the current locale.
slice(): Extracts a portion of a string and returns it as a new string.
split(): Splits a string into an array of substrings.
startsWith(): Checks whether a string begins with a specified substring.
substring(): Extracts the part of a string between the start and end indexes.
toLowerCase(): Converts a string to lowercase letters.
trim(): Removes whitespace from both ends of a string.
toUpperCase(): Converts a string to uppercase letters.

Examples

The following example will concatenate two strings, one lowercase and one uppercase. Try it!


    // user input
    var input = prompt("How is your day?");
    // declare string to add
    var add = " -> That's great! Tomorrow will be an even better one!";
    // show lowercase input + lowercase output
    alert((input.toLowerCase()).concat(add.toUpperCase()))

The next example will show how slice and then trim a string. Try it!


    // declare the string
    var str1 = "Today has been a very long day... I can't wait for tomorrow!";
    // slices string after the ... and to the end of the string, then trims it
    alert(str1.slice(33, str1.length).trim());

You can also loop through strings like arrays, try it!


    var world = "Loop";
    var i;
    // Loop through the world
    for (i = 0; i < world.length; i++) {
        alert(world[i]);
    }
