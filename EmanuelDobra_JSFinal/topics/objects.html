1.6 Objects

Objects in JavaScript are a collection of data. Almost everything is considered an object.
Booleans defined with a new keyword are objects
Numbers defined with a new keyword are objects
Strings defined with a new keyword are objects
Anything apart from primitive data types such as string, number, boolean, null and undefined are objects.

The "this" keyword

The this keyword is a substitute for the new object that will be created when calling the object constructor

The value of this when used in an object, is the object itself after it is declared.


    // creating the user object
    var user = { 
        firstName: "Mike",
        lastName: "Spencer"
        age: 15
        get name() {
            // access this object's name and return it using 'this'
            return this.firstName + " " + this.lastName;
        }
    }

Creating an object

You can also define object constructors and create them using the new keyword. Here is an example of an object constructor and creation


    // constructor for the Student object
    function Student(name, age, id, email) {
        this.name = name;
        this.age = age;
        this.studentId = id;
        this.studentEmail = email;
    }
    // Creating the object
    var mike = new Student("Mike", 19, "10042321", "mike@mail.com"); 

    // you can access any of mike's data by accessing the correct property
    // for example, you can get the age by doing the following:
    var mikeAge = mike.age;

Object Methods

You can also add methods to your object that can be called once the object is created


    // creating the object with a method
    var car = { 
        model: "9000",
        type: "truck",
        color: "red"
        info: function() {
            return this.color + " " this.type + " " + this.model;
            // would return "red truck 9000"
        }
    }

    // calling the objects method
    myCar = car.info;
