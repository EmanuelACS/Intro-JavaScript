1.4 Loops

Loops are used to execute a block of code any number of times. There are 4 types of loops: the for loop, the for/in loop, the for/of loop and the whilefor/of>for/in> loop.

For Loop

The for loop is used to loop through a block of code a number of times, and has the following syntax: 


    for (statement1, statemen2, statement3) {
        // block of code to be executed
    }

Example

Click the button below to try it yourself!


    // ask user how many times he wants to loop
    var question = prompt("How many times do you want to loop?");
    // loop ith times where i is the user's input
    for (int i = 0; i < question; i++) {
        // alert user of the current loop
        if (i == 0) {
            alert("Looping for the 1st time!");
        } else if (i == 1) {
            alert("Looping for the 2nd time!");
        } else if (i == 2) {
            alert("Looping for the 3rd time!");
        } else {
            alert("Looping for the " + i + "th time!");
        }
    }

While loop

The while loop loops through a block of code as long as a condition is true, it has the following syntax: 


    while (condition) {
        // block of code to be executed while the condition is true
    }

Example

Click the button below to try it yourself!


    // tell user keyword to stop loop
    var question = prompt("Enter the keyword 'stop' to end the looping!");
    // as long as user doesn't enter the keyword
    while (question != "stop") {
        // keep prompting the user for input until keyword is given
        question = prompt("Enter the keyword 'stop' to end the looping!");
    }
