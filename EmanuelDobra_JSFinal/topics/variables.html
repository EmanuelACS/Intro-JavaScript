1.1 Variables

JavaScript variables are containers for storing information or data. All variables must have a unique name, called identifiers, which can be short or long. The data it can contain varies from numbers, letters, objects, functions, paths or even specials like null or undefined. Variables can be global or local. Global variables are available to all parts, whereas local are only available in their local scope.

General Syntax

Variables must start with a letter, can contain letters, digits, underscores and dollar signs. In special cases, variables can start with _ or $. You use the keyword var to declare variables. Here are a few examples of normal variables: 


    // this initializes the variable but does not assign a
    // value to it, therefore it remains undefined until changed        
    var declare;
    // To assign text, use quotes " "
    var text = "text";
    // To assign a number, simply add the number
    var number = 7;
    // To assign an expression, add both
    var expression = text + number; 
    // expression would now hold the value "text7"

Methods

You assign values to variables using the "=" operator, which means "assign the value on the right to the variable name on the left"


    var GreetingString = "Hello ";

You can "add" variables together using the "+" operator.


    var thanksForResponding = GreetingString + myName;

Properties

A variable may have properties, which is designed to assist you in adding attributes to that variable. Normally, global variables don't have properites, contrary to ones declare within functions.

Example


    // Asks the user for input 
    var name = prompt("Please enter your name: ");
    var age = prompt("Please enter your age: ");
    var eyeColor = prompt("Please enter your eye color: ");
    var hairColor = prompt("Please enter your hair color: ");
    // Concatenates the strings together and returns the sentence
    function chainVariables(name, age, eye, hair) {
        return "Howdy " + name + ", you've already reached "
            + age + "!\n Your hair is " + hair + "and your eyes are "
            + eye + "!\n It's good to know you better!";
    }
    // Try it!
    alert(chainVariables(name, age, eyeColor, hairColor));
