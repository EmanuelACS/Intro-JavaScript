1.8 Array Object

An array object is a built-in JavaScript object that is used to store multiple values. It is a special variable which can hold more than one value at a time, and each value can be referenced using accessing its index number within the array.

Array Syntax

Declare the array like any other variable using the keyword var and store its values within the curly brackes { }


    // skeleton array declaration
    var array_name = [item1, item2, item3, ...];
    
    // a simple color array
    var colors = ["yellow", "blue", "green", "orange", "red"];

    // a simple number array
    var one_to_five = [1, 2, 3, 4, 5];

You can also declare an array using the new keyword: 


    var colors = new Array("yellow", "blue", "green", "orange", "red");

Accessing Array Elements

To access an element within the array, simply add the index of the item you want. The index of an array starts at 0 for the first item and ends at n-1 where n is the last item's index. Try it!


    // number array
    var one_to_five = [1, 2, 3, 4, 5];
    // access number 3 (whose index is 2)
    var three = one_to_five[2];
    // access last value of array
    var last = one_to_five[one_to_five.length-1];
    alert("My numbers: " + three + " " + last);

You can also access the whole array by calling it normally. Try it!


    // number array
    var one_to_five = [1, 2, 3, 4, 5];
    alert("My array: " + one_to_five);

Changing an Element

Similary to accessing an item, changing it requires assiging it a new value at its index.

Array Methods

A big advantage to using the built-in array object is having access to loads of useful methods. Here are just a few:

concat(): Joins two or more arrays, and returns a copy of the joined arrays
every(): Checks if every element in an array pass a test
findIndex(): Returns the index of the first element in an array that pass a test
forEach(): Calls a function for each array element
indexOf(): Search the array for an element and returns its position
join(): Joins all elements of an array into a string
slice(): Selects a part of an array, and returns the new array
pop(): Removes the last element of an array, and returns that element
push(): Adds new elements to the end of an array, and returns the new length
reverse(): Reverses the order of the elements in an array

Looping over an array

One of the main ways to modify, print or do any actions to an array is by looping through it. Try it!


    // color array
    var colors = ["yellow", "blue", "green"];
    var i;
    // looping and displaying every item of the array
    for (i = 0; i < colors.length; i++) {
        alert(colors[i]);
    }

Examples

In the following example we will create an array, modify it with user input, then sort it and display it.


    // declare array
    var foods_to_love = ["pizza", "chocolate", "candy", "celery", "ribs"];
    // modify array
    foods_to_love[3] = prompt("Who likes celery, enter something you love!");
    // display array
    alert("Here is some of the food you like: " + foods_to_love);

In the following example we turn the array to uppercase using looping and remove the item that doesn't belong


    // declare array
    var fruits = ["orange", "apple", "lime", "banana", "chocolate"];
    var i;
    // looping through the array
    for (i = 0; i < fruits.length; i++) {
        // if we get to chocolate, remove it
        if (fruits[i] == "chocolate") {
            fruits.pop();
        } else {
            // otherwise turn to uppercase
            fruits[i] = fruits[i].toUpperCase();
        }
    }
    alert("Your new array: " + fruits);
